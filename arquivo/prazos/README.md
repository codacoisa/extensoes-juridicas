# Prazos

Script para destacar prazos no Projudi do TJGO de forma visual e facilitar a triagem diária dos processos.

## O que ele faz

- identifica colunas de prazo na página;
- destaca datas próximas com cores diferentes;
- diferencia prazos em fim de semana;
- mostra uma explicação rápida ao passar o mouse;
- mantém filtros e preferências salvos no navegador.

## Requisitos

Você precisa de um gerenciador de userscripts no navegador, como:

- Tampermonkey
- Violentmonkey
- Greasemonkey

## Instalação

Não há canal de instalação mantido para este projeto. Use a extensão ativa [`intimacoes`](../../intimacoes/).

## Onde funciona

O script foi feito para rodar no domínio do Projudi do TJGO.

## Objetivo

A proposta é tornar a visualização de prazos mais rápida e prática dentro da rotina de acompanhamento processual.

## Observação

O destaque visual serve como apoio operacional. A conferência do prazo continua dependendo da análise do caso concreto.
