# Processos Favoritos

Script para marcar processos favoritos no Projudi do TJGO e facilitar o acesso rápido aos casos mais recorrentes na rotina de trabalho.

## O que ele faz

- permite adicionar ou remover processos favoritos;
- destaca os processos marcados na interface;
- oferece um painel para gerenciar a lista de favoritos;
- mantém os dados salvos localmente no navegador;
- permite backup e restauração da lista.

## Requisitos

Você precisa de um gerenciador de userscripts no navegador, como:

- Tampermonkey
- Violentmonkey
- Greasemonkey

## Instalação

Não há canal de instalação mantido para este projeto.

## Onde funciona

O script foi feito para rodar no domínio do Projudi do TJGO.

## Objetivo

A proposta é facilitar a organização e o acesso aos processos acompanhados com mais frequência, tornando a navegação mais prática dentro do sistema.

## Observação

Os favoritos têm função exclusivamente local e organizacional. Eles não alteram o conteúdo processual nem substituem o controle formal dos processos acompanhados.
