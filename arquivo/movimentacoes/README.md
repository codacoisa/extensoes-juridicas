# Movimentações

Script para destacar movimentações processuais no Projudi do TJGO com cores e estilos visuais definidos pelo usuário.

## O que ele faz

- destaca tipos de movimentação com cores diferentes;
- permite personalizar cores, texto, negrito e itálico;
- possibilita ativar ou desativar tipos específicos de movimentação;
- aplica o destaque na listagem principal do processo;
- mantém as configurações salvas no navegador.

## Requisitos

Você precisa de um gerenciador de userscripts no navegador, como:

- Tampermonkey
- Violentmonkey
- Greasemonkey

## Instalação

Não há canal de instalação mantido para este projeto. Use a extensão ativa [`customizacoes`](../../customizacoes/).

## Onde funciona

O script foi feito para rodar no domínio do Projudi do TJGO.

## Objetivo

A proposta é facilitar a leitura do histórico processual, tornando mais rápida a identificação visual das movimentações mais relevantes.

## Observação

O script altera apenas a apresentação visual das movimentações, sem modificar o conteúdo processual.
