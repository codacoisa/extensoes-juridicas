# Destaque Global

Script para destacar termos no Projudi do TJGO de forma visual e facilitar a identificação rápida de informações relevantes na página.

## O que ele faz

- destaca termos definidos pelo usuário em toda a página;
- permite configurar os destaques em um painel próprio;
- possibilita adicionar ou remover termos com mais praticidade;
- permite ajustar cor de fundo, cor do texto e estilo do destaque;
- mantém as configurações salvas no navegador.

## Requisitos

Você precisa de um gerenciador de userscripts no navegador, como:

- Tampermonkey
- Violentmonkey
- Greasemonkey

## Instalação

Não há canal de instalação mantido para este projeto.

## Como usar

O painel do script pode ser aberto pelo atalho **Ctrl + Shift + H**.

## Onde funciona

O script foi feito para rodar no domínio do Projudi do TJGO.

## Objetivo

A proposta é tornar mais rápida a localização visual de nomes, termos e informações importantes dentro da rotina de acompanhamento processual.

## Observação

O destaque visual serve como apoio operacional e organizacional. A análise do conteúdo continua dependendo da conferência do usuário.
